学生端
--------

.. image:: imgs/student-dashboard.png
   :alt: 学生中心截图
   :width: 600

主要流程

1. 打开系统首页；首次登录会自动进入设置向导。
2. 按提示设置 Passkey 或 TOTP，并完善个人信息（如希望使用密码登录需确认管理员允许）。
3. 在“学生中心”提交竞赛记录：填写竞赛名称、分类、角色与学时，上传证明附件（PDF/图片）。
4. 提交完成后可在“我的记录”查看审核状态、推荐学时与附件。
5. 若需导出个人劳动认定表，前往“导出中心”，输入学号即可获取 PDF。

教师视角
--------

.. image:: imgs/teacher-review.png
   :alt: 教师审核界面
   :width: 600

功能点

- 所有教师均可使用 Passkey/TOTP 登录，不允许密码。
- 访问“审核中心”，按照筛选器筛选待审核记录（可按竞赛名称/年份/角色/状态等）。
- 双击表格或点击“审核”打开审查抽屉，可选择初审/复审、填写学时、上传签名图片。
- 使用“批量审核”对多条记录统一提交判断。
- 导出时在“导出中心”确认记录详情与附件，分页预览后再下载 PDF/Excel。

审核人员视角
--------------

.. image:: imgs/reviewer-dashboard.png
   :alt: 审核人员界面
   :width: 600

职责

- 所有审核人员均可使用 Passkey/TOTP 登录，不允许密码。
- “审核中心”显示未审核记录的详细信息：学生信息、附件、竞赛名称、推荐学时。
- 审核时需上传签名（“上传审核签名”区域），每次审核可附带图片。
- 设置“匹配状态”与“审核结果”，系统会根据规则自动计算推荐小时。
- 通过导出模板，生成劳动教育学时认定表并附上签名图片。

管理员视角
-------------

.. image:: imgs/admin-dashboard.png
   :alt: 管理后台
   :width: 600

管理任务

- “管理台”包含学生名单、竞赛库、模板配置、数据导入与删除工具。
- 在“学生名单管理”可筛选/批量删除/重置密码，并控制是否允许密码登录。
- “模板配置”允许上传 Excel 模板、定义导出占位符、设置导入字段映射。
- “数据导入”支持批量导入学生、竞赛与志愿记录，导入时可选择工作表与字段映射。
- “设备与认证”页面显示 Passkey/TOTP 设备、恢复码与一次性重置码。
